# FAQ Accordion

A clean and responsive FAQ Accordion built as a Frontend Mentor challenge using HTML, CSS, and JavaScript.

## 🚀 Features

- Responsive layout
- FAQ accordion functionality
- Toggle answers on click
- Dynamic plus/minus icon switching
- Semantic HTML
- Clean and organized CSS
- Vanilla JavaScript (no libraries)

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 What I Learned

While building this project, I practiced:

- DOM selection with `querySelectorAll()`
- Event handling using `addEventListener()`
- Looping through elements with `forEach()`
- DOM traversal using `parentElement`
- Toggling classes with `classList.toggle()`
- Checking state using `classList.contains()`
- Updating image sources dynamically
